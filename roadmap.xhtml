The PaySwarm web platform is an open standard that enables web browsers and web devices to perform micropayments and copyright-aware, peer-to-peer digital media distribution.

PaySwarm Roadmap

The PaySwarm roadmap outlines all of the expected milestones that must be completed in order for the PaySwarm specification to become an official World Wide Web Consortium standard. The timeline also outlines what it would take for the standard to become successful in the marketplace. Items are struck from the list as they are completed.

The PaySwarm 1.0 Standardization Process

  • Implement complete reference implementation of PaySwarm 1.0 spec. -- November 30th, 2009
  • Complete first draft of PaySwarm Use Cases document. -- December 24th 2010
  • Finish first draft of PaySwarm FAQ. -- January 15th, 2010
  • Release reference implementation as open source. -- February, 1st, 2010
  • Release short introductory video for PaySwarm technology. -- February 22nd, 2010
  • Create PaySwarm community mailing lists, IRC channels and Twitter streams. -- March 2nd, 2010
  • Gather support in music, newspaper and television industries to standardize work at W3C. -- ongoing
  • Create a PaySwarm Working Group at the World Wide Web Consortium.
  • Publish the PaySwarm Use Cases document as a W3C Note.
  • Publish the First Public Working Draft of PaySwarm Web API 1.0.
  • Publish the First Public Working Draft of Web API Discovery 1.0.
  • Refine and extend PaySwarm to meet the digital content industry's business needs.
  • Publish the Last Call for PaySwarm Web API 1.0.
  • Publish the Last Call for Web API Discovery 1.0.
  • Refine documents under guidance from digital content industry.
  • Enter Candidate Recommendation phase for PaySwarm Web API 1.0 and Web API Discovery 1.0.
  • Call for two interoperable implementations begins.
  • Digital Bazaar will provide the first implementation.
  • Another individual or organization will provide the second implementation.
  • Enter Proposed Recommendation phase for PaySwarm Web API 1.0 and Web API Discovery 1.0.
  • Publish PaySwarm Web API 1.0 and Web API Discovery 1.0 as a W3C Recommendation

PaySwarm 1.0 in the Marketplace

  • Help browser vendors to implement the Buyer and Seller portions of the PaySwarm standard.
  • Help music players to implement Buyer/Seller portions of the PaySwarm standard.
  • Help other open-source projects to implement Buyer/Seller portions of the PaySwarm standard.
  • Gather feedback from performance and experience in the marketplace for PaySwarm 2.0 Working Group.
